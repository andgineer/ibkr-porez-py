# Встановлення
Для встановлення рекомендовано використовувати `uv tool` від Astral, щоб уникнути конфліктів із системними пакетами Python.

### Встановлення `uv`
[Встановлення `uv`](https://docs.astral.sh/uv/getting-started/installation/)

### Встановлення застосунку

```bash
uv tool install ibkr-porez
```
